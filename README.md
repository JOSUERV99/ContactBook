## Instalation

- You need to have mongo server running in your computer, if not are running try with 
```bash
mongod --dbpath <path>
 ```` 

Use npm to:

Install express (only if you don't have)

```bash
    npm i express
```
Download the dependencies:

```bash
    npm install
```

## Running the project


```bash
    npm start 
```


```bash
    npm run dev # with nodemon for coding!
```
